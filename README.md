# dbservice

> This is a Database microservice project developed using AWS DocumentDB

## System requirements

```sh

# node version
    - 10+

# npm version
    - 6+

```

# installation

```sh

# clone the repository
    - git clone git@github.com:Ashwinkumar-R/dbservice.git

# cd to project
    - cd dbservice

# npm install
    - npm install

# start

    - node main.js <arguments>

```
